Ubuntu环境安装Nodejs及Express

发表于 2014-02-19 | 分类于 nodejs

没用apt-get，原因是有人说版本不兼容，所以没有尝试

1、git clone nodejs源码

$ git clone git://github.com/joyent/node.git

2、编译安装

$ cd node
$ ./configure
$ make
$ sudo make install

3、这时候nodejs 和 npm 已经都装好了，可以尝试看下版本号，确保命令可用

$ node -v
$ npm -v

4、安装express模块（web开发框架）

$ sudo npm install express -g

-g 的意思是把这个模块安装到全局模块，切记要加上，这样在任何位置下都可以使用express这个命令构建express项目

5、构建express项目

$ cd ~/node-project
注：这是我自己的项目目录，你们可以根据自己的需要修改
$ express node-first
注：node-first是项目名称，执行之后会生成一个node-first的文件夹
$ cd node-first
$ sudo npm install
$ node app

OK，基本上已经完成
打开浏览器访问localhost:3000看看效果吧
